Google Sheets — Base de Datos

Todos los cambios se sincronizan automáticamente

Cómo configurar (solo una vez):

  1. Abre Google Sheets y crea una hoja nueva
  2. Ve a Extensiones → Apps Script
  3. Borra el código existente y pega el contenido de apps-script.gs
  4. Clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona
  6. Copia la URL generada y pégala abajo · Guarda

Instalar como Aplicación

Usa InvenHogar sin abrir el navegador

Android (Chrome)

  • 1 Abre InvenHogar en Chrome
  • 2 Toca el menú (⋮) en la esquina
  • 3 Selecciona "Agregar a pantalla de inicio"

PC (Chrome / Edge)

  • 1 Busca el ícono en la barra
  • 2 O abre el menú (⋮) → "Instalar"
  • 3 Se abrirá como aplicación de escritorio

iPhone / iPad

  • Abre InvenHogar en Safari
  • Toca el botón Compartir (cuadro con flecha)
  • Selecciona "Agregar a pantalla de inicio"
